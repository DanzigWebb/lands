DESCRIPCIÓN

Es un sello de cejas fácil de usar que crea cejas arqueadas de manera uniforme y natural.

El dispositivo de belleza consta de una base de plástico a la que se une una esponja con forma de ceja de mujer. Debe aplicarse a la ceja y presionarse. Después de un par de segundos, se puede quitar el sello.
